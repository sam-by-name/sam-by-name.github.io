How does JavaScript compare to HTML and CSS?

If HTML is your skeleton and CSS your external cosmetic complexion, then JavaScript is your behavior. HTML and CSS are constants but through JavaScript we can augment and alter them to seem moving and alive. Or even disappear them completely.

Explain the difference between accessing data from arrays and objects.

An array has a number based indexing system, so you call upon its contents by using the array name followed by its numerical index [42]. An objects properties on the other hand are defined and called upon by their property names.
A simple way to think about how they differ is, one is a nice orderly row of items (array) and can be accessed by number. The other is an unsightly pile of items (object) and to know whats in the pile and how to access them, you shall name them and call for them by name.
