Tech Blog 1

Q.
What is grid based design?

A.
The grid based design has been formulated to be a simple to implement, unseen structure, that helps a web developer quickly arrange their page and the placement of their content.

Q.
What's all the hype about responsive webpages?

A.
A responsive webpage, is one that is aware of the screen resolution and ratio, that it is going to be displayed on, and is coded such that it alters how and what it shows, to best highlight the content of the page accordingly.

Q.
What is semantic structure?

A.
Semantically structuring your HTML and CSS makes it such that strangers to the code can understand what content will go where, when it get's opened in a browser.
i.e. You wrap the different segments of your content with descriptive tags.

Any other points you found interesting and want to reflect on.
