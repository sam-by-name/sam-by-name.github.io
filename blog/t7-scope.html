Scope

What 'Scope' is, and how it works in JavaScript.

Well dear friend 'Scope' as used inside JavaScript is actually quite similar to how we use it in our language and day to day lives. Of-course it has its own quirks and so now I shall elaborate. In JS there are two types of 'Scope'

  • Global scope
  • Local scope
'Global Scope' is the entirety of your code (HTML/CSS included) and so any function or variable declared in the 'global scope' of your JS is visible anywhere inside your code. 'Local scope' is the code inside your 'Global functions' that have been written inside the 'Global Scope'.
All variables declared inside the 'Local Scope' are only visible inside the 'Local scope' ('var' scat, no-one invited you). 'Local' can see the 'Global' but not the other way around.
A decent analogy is one of 'House rules' vs 'Societal rules', It may very well be considered a kindness (or even mandatory) to hurl your food at other people inside your home.
But I reckon that those house rules will not hold up so well outside of your home.
And now a technical example : 
            // This is the 'Global Scope' and everything that gets declared out here is visible and accessible everywhere.
            
            let global = "Thou shall not kill";    // I am a global variable

            function globalFunction () {
              let local = 'Throwing food is fun';  // I am a local variable
              console.log(local);                  // Calling the variable 'local' here works 
              console.log(global);                 // The same goes for 'global'
            }
             
            console.log(local);                    // The variable 'local' does not exist here because it was 
            declared inside 'globalFunction()' and as such is inside local scope.

            console.log(global);                   // The variable 'global' works here and so the console will print
            'Thou shall not kill'.

            console.log(globalFunction());         //The function declaration is in global scope and so its contents will now get printed
            'Throwing food is fun'
            'Thou shall not kill'
Another analogy would be: If a car has heavily tinted windows, its occupants can freely see the world around them without what happens inside being visible to everyone outside of the car.
