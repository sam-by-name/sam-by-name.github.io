Parameters Defined

Parentheses ( )

  • Are used to define functions function newFunction ()
  • Or to call a function newFunction ()
  • For grouping to dictate the Order of Operations statements first (2 + (2 * 6))
  • Can be used to define a for/while/do loop for (let i = 0; i < 5; i++)

Brackets [ ]

  • For creating arrays var array = firstArray [1, two, "three"]
  • Addressing an arrays index alert(firstArray[2])
  • Accessing object properties firstObject["property"]

Braces { }

  • They are used for control flow if (1+2 === 5-2) {alert("Yay for math")} else {alert("Error"}
  • To enclose blocks of code like a function definition function myFunc() {"function definition here"}
  • When creating an object var trump = {language: "baby", iq: "less than ideal", empathy: 0}

Single quotes ' ' and Double quotes " "

  • Single and double quotes are interchangeable in JavaScript, but if you use single quotes inside double quotes (or vice-versa) then they don't need to be escaped in some circumstances.

    But it can still get a bit messy
    alert('Life isn\'t "always" as it seems')
