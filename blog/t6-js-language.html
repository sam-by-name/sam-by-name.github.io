parentheses ( )

  • Are used to define functions function newFunction ()
  • Or to call a function newFunction ()
  • For grouping to dictate the Order of Operations statements first (2 + (2 * 6))
  • Can be used to define a for/while/do loop for (let i = 0; i < 5; i++)

brackets [ ]

  • Creating arrays var array = firstArray [1, two, "three"]
  • Addressing an arrays index alert(firstArray[2])
  • Accessing object properties firstObject["property"]

  • creating Objects
  • opening of a block
